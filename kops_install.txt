🚀 kops Installation 🚀


🧩 create an instance
🧩 install kubectl
🧩 curl -LO "https://dl.k8s.io/release/$(curl -L -s https://dl.k8s.io/release/stable.txt)/bin/linux/amd64/kubectl"
   chmod +x kubectl
   sudo mv kubectl /usr/local/bin  (to access kubectl)
   kubectl version (to verify)

🎯 install kops 
   curl -Lo kops https://github.com/kubernetes/kops/releases/download/$(curl -s https://api.github.com/repos/kubernetes/      kops/releases/latest | grep tag_name | cut -d '"' -f 4)/kops-linux-amd64
   chmod +x kops
   sudo mv kops /usr/local/bin/kops

🧩 aws configure (if its not installed)
  install awscli
  sudo apt install awscli
  aws configure

🧩 create s3 bucket

   aws s3 mb s3://bucketname --region us-east-1  (mb-make bucket, bucket name should be globally unique)
   aws s3 ls (to see buckets)

🧩 create cluster
  kops create cluster --name=kop.local.com \
  --state=s3://bucketname \
  --zones=us-east-1a \
  --node-count=2 \
  --node-size=t3.micro \
  --master-size=t3.micro \
  --master-volume-size=8 \
  --node-volume-size=8 

   (this is for 2 workers and 1 master)

🧩 Running the above command will generate a preview of all the resources that Kops will create. Once you are okay with the      preview, run the below command to create those resources. Update the S3 bucket to your S3 bucket.

    export KOPS_STATE_STORE=s3://bucketname

🧩 to validate 
   kops validate cluster clustername

    then kubectl get nodes

🧩 to delete
    kops delete cluster cluster-name
